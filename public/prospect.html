Billing Details

Blockforger uses a token-based billing system for usage-based pricing.

LLM Operations: 10 tokens per LLM schema generation or JSON object generation call.

Page Loads: 1 token minimum, plus 1 token per 1MB of data loaded.

Storage: 1 token per 10MB of storage used (billed daily).

Payment: Purchase tokens through our Stripe integration. Tokens never expire.

For detailed billing information and account management, please create an account.
